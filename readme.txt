== Description ==

This plugin allows you to append a login or logout link depending if user is logged in at time.
You can choose which menu to append the links to via the Menu setting in the customizer.

### Features
* Easily display login/logout link on a menu
* Simple dropdown in the menu to choose the location

== Installation ==

### Plugin Installation
1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress

== Frequently Asked Questions ==

= Can I choose multiple menus? =

No. At this stage it will only allow you to select one menu.

= Can I redirect after logging out? =

No. At this stage it will take the user to the login screen.
